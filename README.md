# Zephyr-Wabot
Bot WhatsApp berbasis **Baileys** yang dibuat untuk otomatisasi chat, fitur grup, dan pengelolaan pesan.  
Dikembangkan oleh [RullzFuqi](https://github.com/RullzFuqi).

---

## 📦 Instalasi

Salin dan jalankan perintah di bawah ini di Termux, VPS, atau Linux kamu:

```bash
pkg install git ffmpeg -y
git clone https://github.com/RullzFuqi/Zephyr-Wabot
cd Zephyr-Wabot
npm install
npm start
