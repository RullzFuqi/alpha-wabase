# Zephyr-Wabot
**Zephyr-Wabot** adalah bot WhatsApp berbasis [Baileys](https://github.com/WhiskeySockets/Baileys) yang dirancang untuk otomatisasi percakapan, manajemen grup, broadcast, dan berbagai fitur interaktif.  
Dikembangkan oleh [RullzFuqi](https://github.com/RullzFuqi).

---

## 🧩 Persyaratan Dasar
Pastikan sistem kamu sudah memiliki:
- Node.js versi **18 atau lebih baru**
- Git
- FFmpeg (untuk fitur media)

Jika belum, ikuti langkah di bawah.

---

## ⚙️ Instalasi

### 1. Instal dependensi sistem
```bash
pkg install git ffmpeg -y

2. Clone repositori

git clone https://github.com/RullzFuqi/Zephyr-Wabot

3. Masuk ke folder proyek

cd Zephyr-Wabot

4. Instal semua modul Node.js

npm install

5. Jalankan bot

npm start
```

---

🧠 Fitur Utama

🔑 Login menggunakan pairing code (tanpa QR)

🧭 Menu interaktif dengan tombol (button & list)

🏷️ Sistem log CMD real-time

👥 Event grup (welcome, leave, dsb)

📢 Broadcast dan push message

🛍️ Store menu dengan testimoni & transaksi

⚡ Auto reconnect dan notifikasi error



💡 Tips Penggunaan

Jika koneksi gagal, hentikan proses (CTRL + C) lalu jalankan kembali npm start.

Untuk mengganti nomor pairing, hapus file sesi (session atau auth_info.json).

Jalankan bot di Termux, VPS, atau Panel Pterodactyl dengan koneksi stabil.
